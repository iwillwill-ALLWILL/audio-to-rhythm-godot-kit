# RhythmKit Godot Runtime Addon

This addon is a reference runtime for bundles generated by `audio-to-rhythm-godot-kit`.

## Use

Copy these folders into your Godot project:

```text
addons/rhythmkit/
levels/<song_id>/
```

Instantiate the default view:

```gdscript
var view = preload("res://addons/rhythmkit/RhythmGameView.tscn").instantiate()
add_child(view)
view.load_bundle("res://levels/<song_id>", "normal")
view.start_game()
```

Or use only the loader and build your own visuals:

```gdscript
var loader = preload("res://addons/rhythmkit/RhythmLevelLoader.gd")
var chart = loader.load_bundle("res://levels/<song_id>", "hard")
```

The host game owns final UI, art, scoring flavor, progression, and theme. This addon only demonstrates loading, timing, note spawning, and judgement.
