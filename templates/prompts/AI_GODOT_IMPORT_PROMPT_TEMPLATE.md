# AI Prompt Template: Make a Godot Game Import RhythmKit Bundles

Use this prompt inside the user's existing Godot 4 project. Replace placeholders before sending to Codex/Claude Code/Cursor/Hermes.

```text
You are working inside my existing Godot 4 project.

I have a RhythmKit level bundle generated by audio-to-rhythm-godot-kit.
Bundle location after copy:
  res://levels/{SONG_ID}/

The bundle contains:
  metadata.json
  audio/song.wav
  charts/easy.chart.json / normal.chart.json / hard.chart.json
  integration/README.md

Your task: make this Godot project able to import/play the rhythm bundle without breaking existing gameplay.

Requirements:
1. Inspect the project structure first. Do not assume scene names.
2. Add reusable rhythm runtime files under res://addons/rhythmkit/ unless an equivalent system already exists.
3. Load res://levels/{SONG_ID}/metadata.json.
4. Expose difficulty selection from metadata.charts.
5. Load selected chart JSON and audio/song.wav.
6. Render placeholder rhythm lanes/notes or map chart notes into the project's existing UI.
7. Use note.time, note.lane, note.type, chart.difficulty.note_speed, perfect_window, good_window.
8. Add a minimal entry point from the existing game: debug scene, menu button, or callable function.
9. Keep art/style replaceable; do not hard-code final assets.
10. Run Godot headless/console to verify no parser/runtime errors, and fix them.

Suggested usage API:
var view = preload("res://addons/rhythmkit/RhythmGameView.tscn").instantiate()
add_child(view)
view.load_bundle("res://levels/{SONG_ID}", "normal")
view.start_game()

Deliver:
- list of files changed;
- how to launch the rhythm level;
- verification command/output.
```
